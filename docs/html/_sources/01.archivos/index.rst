Manejo de archivos
==================
La unidad está dedicada a dos tareas distintas, aunque complementarias:

+ La **gestión** de archivos locales de disco: listado de directorios,
  comprobación de permisos, creación y eliminación, etc.
+ El **acceso y manipulación** de archivos de intercambio de información sin un
  formato estándar, ya que los estándares (|XML|, |JSON|, etc) los dejaremos
  para la unidad posterior.
